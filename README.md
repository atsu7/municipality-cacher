# このアプリについて
GeocachingのFoundデータの日本の市区町村にまつわる統計を表示するウェブアプリです。

Geocaching.comのMy Findsポケットクエリを読み込んで動作します。
動作はフロントエンドのみで行われます。

## 動作
ブラウザ上で市区町村ポリゴンを読み込み、ユーザーがアップロードした GPX の点を各ポリゴンに割り当てて集計・可視化する、完全フロントエンドのアプリです。

全体構成と主要ファイル
- `index.html` 
UI（ファイル選択ボタン、地図領域、都道府県バッジ、ランキング、Only 1F、Challenges、CSV ダウンロードなど）を定義。
外部ライブラリ（Leaflet、turf、rbush、togeojson 等）を読み込み、最後に main.js を読み込む。
- `main.js`
地図初期化、ポリゴン読み込み、GPX パース、点→ポリゴン割当、各種 UI 更新ロジックが入っている。
- `polygon`
市区町村ポリゴン（GeoJSON/TopoJSON）。index.jsonに読み込むファイル名の配列を置く必要がある。

## 市区町村にまつわるマニアックな話
地方自治法では、「基礎的な地方公共団体」として、市町村および特別区が定められています。これらは一般に「市区町村」と総称されます。現在、日本には792の市、743の町、183の村、23の特別区があります。

なお、政令指定都市に設置されている行政区は地方公共団体ではなく、都に設置されている特別区とは法的な位置づけが異なります。また、東京都の島嶼部を除く町村はすべて「郡」に属していますが、郡は現在では行政上の権限を持たず、住所表記などに用いられる地理的区分にすぎません。

本アプリでは、市区町村の区分として、地方自治法における「基礎的な地方公共団体」である市町村および特別区を採用しています😀
